使用matplotlib绘图¶

matplotlib绘图有两种，一种是类似matlab的api绘图，一种是面向对象的api。建议学习并使用matplotlib的面向对象的绘图API。它非常强大。对于具有子图、嵌入式和其他组件的高级图形来说，使用它是非常好的。面向对象编程的主要思想是拥有一个可以应用函数和操作的对象，并且没有任何对象或程序状态应该是全局的(比如类似于matlab的API)。当创建了多个图形，或者图形包含多个子图时，这种方法的真正优势就变得显而易见了。

在python中显示图像，要使用show函数。 在jupyter notebook中当我们使用IPython笔记本时，使用matplotlib后端输出嵌入在笔记本文件中的图形是很方便的。为了激活这个后端，在笔记本的开头添加：%matplotlib inline 还可以使用以下方法激活内联matplotlib绘图：%pylab inline

区别在于%pylab inline将多个包导入全局地址空间(cip，numpy)，同时%matplotlib inline只设置内联绘图。在为IPython1.0+创建的新笔记本中，我建议使用%matplotlib inline，因为它更整洁，而且您可以更好地控制哪些包被导入以及如何导入。通常，枕木和蒙皮是分别进口的，包括： import numpy as np import scipy as sp import matplotlib.pyplot as plt

内联后端有许多配置选项，可以使用IPython魔术命令进行设置。%config更新设置InlineBackend。例如，我们可以切换到SVG图形或更高分辨率的图形，其中之一是：%config InlineBackend.figure_format='svg'或：%config InlineBackend.figure_format='retina'

有关更多信息，请键入： %config InlineBackend

使用figure创建画布并绘制图像¶

使用figure只是单纯的创建一个画布，但是并没有子图对象。要在画布上添加子图，需要使用fig.add_axes函数，并且可以指定图像在画布中的位置。

In [21]:
x = np.linspace(0, 5, 10)
y = x ** 2
fig=plt.figure(figsize=(4,4))
axes1=fig.add_axes([0.1,0.1,0.8,0.8]) # left, bottom, width, height (range 0 to 1)
axes2 = fig.add_axes([0.2, 0.5, 0.4, 0.3]) 

axes1.plot(x, y, 'r')
axes1.set_xlabel('x')
axes1.set_ylabel('y')
axes1.set_title('title')

axes2.plot(y, x, 'g')
axes2.set_xlabel('y')
axes2.set_ylabel('x')
axes2.set_title('insert title');

可以看到fig.add_axes可以设置左轴和底轴的位置和长度，范围为0到1。画布是有大小的，前两个参数的含义代表画布的左分位点和下分位点的位置。在该位置防止轴，后两个设置轴的长度。所以一般对图像的位置有较为严格要求的使用figure和add_axes搭配使用。

使用subplot¶

subplot有三个参数，一个是行数，一个是列数，还有一个是创建子图的位置。这里的位置是第几个，而不是下标。按照从左到右，从上到下的顺序。 创建之前需要先使用figure创建一个画布。

使用subplots¶

subplots可以通过figsize设置大小，设置行数，设置列数。subpolts同时创建了画布和子图，可以同时创建两个对象。可以通过子图对象的下标确定子图的位置，也可以使用子图对象，然后使用fig.tight_layout方法自动排列。该方法自动调整图形画布上轴的位置，以便不存在重叠内容。 子图对象也可以当成一个迭代器，对里面的每一个子图进行操作。

In [27]:
fig, axes = plt.subplots(nrows=1, ncols=2)

for ax in axes:
    ax.plot(x, y, 'r')
    ax.set_xlabel('x')
    ax.set_ylabel('y')
    ax.set_title('title')
    
fig.tight_layout()

多图-在一块画布中绘制大小不一的子图¶

使用subplot2grid函数，来管理子图的布局。有两个参数，一个是网格的行列，用元组展示，然后决定起始位置，也是用元祖表示。从起始位置决定向右还是向下合并网格。colspan向右合并，rowspan向下合并。

In [33]:
fig = plt.figure()
ax1 = plt.subplot2grid((3,3), (0,0), colspan=3)
ax2 = plt.subplot2grid((3,3), (1,0), colspan=2)
ax3 = plt.subplot2grid((3,3), (1,2), rowspan=2)
ax4 = plt.subplot2grid((3,3), (2,0))
ax5 = plt.subplot2grid((3,3), (2,1))
fig.tight_layout()

使用add_gridspec（行，列）函数，创建网格。然后利用add_subplot(网格的一部分，切片)来创建子图。然后在子图上作图即可。

In [34]:
fig=plt.figure(constrained_layout=True,figsize=(8,5))#创建画布和确定大小
plt.subplots_adjust(hspace=1,wspace=1.1)#调整图形之间的长宽
gs=fig.add_gridspec(4,7)#添加网格
#下面为绘制子图并确定子图大小
f1=fig.add_subplot(gs[:-1,0:3])
f2=fig.add_subplot(gs[:-1,3])
f3=fig.add_subplot(gs[:-1,4:7])
plt.plot(x,y,'r')
f4=fig.add_subplot(gs[-1,0:3])
f5=fig.add_subplot(gs[-1,4:7])

使用matplot.gridspec模块中的GridSpec函数。设定行，设定列，设定行的切分，设定列的切分，然后两两组合，将网格分成切割好的部分，然后再利用迭代和fig.add_subplot函数来生成子图。

In [36]:
#网格规范
import matplotlib.gridspec as gridspec
fig = plt.figure()

gs = gridspec.GridSpec(2, 3, height_ratios=[2,1], width_ratios=[1,2,1])
for g in gs:
    ax = fig.add_subplot(g)
    
fig.tight_layout()

只要是与文字有关的都可以被字体的设置应用。可以先使用全局字体大小和字体系列，这适用于图形中的所有文本元素(勾选标签、轴标签和标题、图例等)： # Update the matplotlib configuration parameters: matplotlib.rcParams.update({'font.size': 18, 'font.family': 'serif'})

图例¶

更好的方法是使用label="label text"关键字参数时，将绘图或其他对象添加到图形中，然后使用legend方法将图例添加到图中： 这种方法的优点是，如果从图形中添加或删除曲线，则图例将自动更新。

In [41]:
ax.plot(x, x**2, label="curve1")
ax.plot(x, x**3, label="curve2")
ax.legend();

这个legend函数采用可选的关键字参数。loc它可用于指定在图形中绘制图例的位置。的允许值loc都是不同地方的数字代码，图例可以画出来。

In [52]:
ax.legend(loc=0) # let matplotlib decide the optimal location
ax.legend(loc=1) # upper right corner
ax.legend(loc=2) # upper left corner
ax.legend(loc=3) # lower left corner
ax.legend(loc=4) # lower right corner
# .. many more options are available

对数标度与科学符号¶

还可以为一个或两个轴设置对数标度。这个功能实际上只是Matplotlib中更通用的转换系统的一个应用程序。 每个轴的刻度分别使用set_xscale和set_yscale接受一个参数(在本例中为“log”)的方法。标度变成了10的n次方

In [66]:
fig, axes = plt.subplots(1, 2, figsize=(10,4))
      
axes[0].plot(x, x**2, x, np.exp(x))
axes[0].set_title("Normal scale")

axes[1].plot(x, x**2, x, np.exp(x))
axes[1].set_yscale("log")
axes[1].set_title("Logarithmic scale (y)");

由于轴上有大量的数字，它通常更好地使用科学符号：

设置ticks和ticks的标签¶

我们可以显式地确定我们想要的轴的位置。set_xticks和set_yticks，这两种方法都采用一个值列表，用于将滴答放在轴上的位置。我们也可以使用set_xticklabels和set_yticklabels方法为每个滴答位置提供自定义文本标签列表：

In [68]:
fig, ax = plt.subplots(figsize=(10, 4))

ax.plot(x, x**2, x, x**3, lw=2)

ax.set_xticks([1, 2, 3, 4, 5])
ax.set_xticklabels([r'$\alpha$', r'$\beta$', r'$\gamma$', r'$\delta$', r'$\epsilon$'], fontsize=18)

yticks = [0, 50, 100, 150]
ax.set_yticks(yticks)
ax.set_yticklabels(["$%.1f$" % y for y in yticks], fontsize=18); # use LaTeX formatted labels
